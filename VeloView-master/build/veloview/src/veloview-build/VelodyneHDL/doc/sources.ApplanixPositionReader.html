Applanix Position Reader (ApplanixPositionReader)

Read Applanix data files.
Read Applanix data files.
Property Description Default(s) Restrictions
FileName This property specifies the file name for the reader.

The value(s) must be a filename (or filenames).
BaseYaw This property specifies the "base" yaw orientation of the GPS sensor. The value is subtracted from the reported yaw to get actual yaw. 0.0
BaseRoll This property specifies the "base" roll orientation of the GPS sensor. The value is subtracted from the reported roll to get actual roll. 0.0
BasePitch This property specifies the "base" pitch orientation of the GPS sensor. The value is subtracted from the reported pitch to get actual pitch. 0.0
